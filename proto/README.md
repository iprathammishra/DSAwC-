## Steps to create the executable:

0. Make the `CMakeLists.txt` file.

1. Make a separate directory.

```bash
mkdir build
```

2. Change directory.

```bash
cd build
```

3. Initiate `cmake`.

```bash
cmake ..
```

4. Make the executable.

```bash
cmake --build .
```

5. Executable will be present in the `Debug/` folder.
